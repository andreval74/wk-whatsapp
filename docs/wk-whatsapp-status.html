O que já existe hoje

Parser de export parseWhatsAppExport()
Lê arquivos .zip/.txt exportados do WhatsApp (via fflate no navegador) e extrai data, hora, remetente e texto de cada linha.
Agrupamento por dia groupIntoDayBlocks()
Organiza as mensagens em blocos por data, base para toda a navegação do feed.
Categorização automática autoCategorize()
Heurística de frequência de palavras (estilo TF-IDF) que detecta os assuntos sozinha, sem IA de verdade.
Busca por IA em linguagem natural parseAIQuery()
Reconhece datas, valores, remetente e assunto em perguntas livres como "pagamentos acima de 1000 em 2025".
Pesquisas salvas
Guarda buscas favoritas no localStorage do navegador para reuso rápido.
100% estático
Sem backend: todo o processamento roda no navegador, servido apenas como arquivos estáticos pelo XAMPP.

O que estamos construindo agora

Backend Node.js separado server/
Processo novo, independente do XAMPP/PHP, responsável só pela conexão com o WhatsApp.
sessionManager.js
Gerencia múltiplas sessões simultâneas. Qualquer pessoa pode conectar sua própria conta via QR, isoladas entre si.
whatsappService.js (Baileys)
Fala o protocolo real do WhatsApp via WebSocket (Baileys 6.7.23, estável), sem precisar de navegador/Chrome.
API HTTP api.js
Endpoints para criar sessão, checar status, listar contatos reais e buscar mensagens de um contato.
WebSocket de QR/status liveSocket.js
Empurra o QR code (como imagem) e mudanças de status para o navegador em tempo real.
Frontend live.js
Botão "Conectar WhatsApp", modal de QR, lista de contatos reais. Reaproveita 100% do agrupamento/categorização já existente.

Decisões já confirmadas

  • Somente leitura: o sistema nunca envia mensagens nem altera nada no WhatsApp do usuário.
  • Qualquer pessoa pode conectar a própria conta via QR, não é uso restrito a um único dono.
  • Histórico: sincroniza o que o WhatsApp entregar + combina com o export já importado, via mergeMessages() (dedup por data/hora/remetente/texto).
  • Biblioteca: @whiskeysockets/baileys na versão estável 6.7.23 (não a pré-lançamento 7.0.0-rc).
  • Suporte a múltiplas sessões simultâneas, uma por pessoa/QR.
